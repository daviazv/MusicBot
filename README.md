## Music Bot
Um bot de música **simples, rápido e fácil de usar** para servidores do Discord.  
Ideal para quem quer tocar músicas sem complicação.

## Configuração
1. Abra o arquivo **`.env`**
2. Adicione o token do seu bot:

``DISCORD_TOKEN=seu_token_do_bot_aqui``

## Instalação
No terminal, dentro da pasta do projeto, execute:

``pip install -r requirements.txt``

## Iniciando o bot
Depois de instalar as dependências, inicie o bot com:

``python main.py``
